Pegasys News and Announcements

PBS Pegasys Users:

The FY 2014 Appropriations Bill split the PBS Building Operations budget account into two separate program codes (also known as budget activities). As a result of this change, a new program code (PG60) was added to Pegasys.

Pegasys transactions for PBS Building Operations are now recorded with one of the following:

  • Program code (PG61) used for Building Services transactions.
  • program code (PG60) used for Salaries and Expenses transactions.

Click to view a crosswalk, which identifies the transactions that should use code PG61 and those that should use PG60.

If you have questions, please contact Dawn Stalter at dawn.stalter@gsa.gov or (202) 205-8921.

AGENCY CODE CHANGES IN PEGASYS DUE TO BAAR PHASE 2A IMPLEMENTATION

Pegasys now identifies an agency with a three-digit code. This transition to using a three-digit code to identify agencies is a part of the Government-wide Treasury Account Symbol Adjusted Balance System (GTAS) project and was part of the recent implementation of the Phase 2A Billings and Accounts Receivable (BAAR) module.

To find an agency’s new identification code, visit https://insite.gsa.gov/portal/content/529890, and review the document “2-Digit to 3-Digit Agency Code Crosswalk.”

EASi Interface implemented with Pegasys

The first phase of the Enterprise Acquisition Solution integrated (EASi) interface was implemented with Pegasys on the weekend of November 1. As a result of this interface implementation, Pegasys users will now be limited to view-only permission to the following document types:

  • EF - EASI Reservation
  • EQ - EASI Purchase Request
  • EP - EASI Stand-Alone and Task Order
  • EJ - EASI Intra-Governmental Obligation
  • EK - EASI Recurring Service without LoGO clause Obligation
  • EN - EASI Construction Obligation

No other permissions are allowed beyond view-only because these documents are now coming through the EASi interface. Requests for any other type of permissions to these document types will be denied.

For more information, visit https://insite.gsa.gov/portal/content/692858
