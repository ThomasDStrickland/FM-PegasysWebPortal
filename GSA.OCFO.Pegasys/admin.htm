Pegasys Support >> Administration

Access to Pegasys as well as the Roles and Approval Types needed to perform work in the system are limited to those users who have been authorized by their supervisor and their Pegasys Functional Coordinator or Service Representative. To add new users or change the roles and approval types of current users, a User ID Request form must be completed.

We also have several queries available so you can check which Roles and Approval Types a user already has. You can find definitions for the most common Roles and Approval Types.
