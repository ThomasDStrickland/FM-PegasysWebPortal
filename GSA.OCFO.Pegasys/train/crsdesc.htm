Pegasys Support >> Training >> Course Descriptions

Credit Cards:

The Credit Card class is intended for credit card holders, reconcilers, and approving officials. The Pegasys Credit Card class details the concepts and functions and provides step-by-step instructions of the Pegasys Credit Card subsystem.

Direct Pay

The Direct Pay class is intended for associates who are processing oral procurements. Using a direct pay invoice form allows the transaction to be processed and paid without a corresponding receipt or order.

Estimated Accrual

The Estimated Accruals class is intended for those who want to record expenses in a timely fashion. We will focus on processing three types of estimated accrual forms: 3-way Order–Invoice–Receipt, Self-Reversing 2-way Order–Invoice, and the No Way Intra-Agency.

External Inter-agency & Intra-agency

The External Inter-agency & Intra-agency class is intended for users who process MOU’s, either Inter-agency or Intra-agency. Inter-agency Agreements are obligations incurred to procure goods or services between other federal agencies and GSA. Intra-agency Agreements are obligations incurred to procure goods or services between GSA services.

FMIS (Central Office)

The FMIS Web Basics class is an introductory class to help new users become more comfortable with the Pegasys query tool as well as make current users more aware of the options in the system. FMIS menu items, query grids and reports are among the topics discussed.

FMIS (NCR)

The FMIS Web Basics class is an introductory class to help new users become more comfortable with the Pegasys query tool as well as make current users more aware of the options in the system. FMIS menu items, query grids and reports are among the topics discussed.

Pegasys 101

Pegasys 101 is an introductory class that gives a basic overview of the system, reviews the Pegasys homepage, and discusses how to navigate within the system. The class is minimally hands-on for users that have a Pegasys User ID and an active account.

Purchase Requests

The Purchase Request class is intended for those who request goods or services. This class will also cover approvals and routing documents.

Purchase Orders

The purchase order class is intended for contracting officers and authorized buyers. It describes the process of generating purchase orders (GSA Form 300) in Pegasys, which obligates funds and informs the vendor of our requirements.

Receipts

The Receipt classes is intended for those receiving goods or services. Receipt forms record the delivery and acceptance of goods or services.
