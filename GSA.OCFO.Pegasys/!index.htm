PEGASYS PLANNED OUTAGE:

Pegasys will be unavailable so that USDA's Pegasys Financial Services Team can complete BAAR Phase 3 Wave 3.3 implementation. The implementation tasks require coordination across multiple teams totaling close to 50 people. The system must be made unavailable to keep data static and to perform system validation before users access the system.

AFFECTED AREA

All Pegasys users and all Pegasys functions will be affected. All users of CGE Travel, CCRC, FMIS, Business Objects and VCSS will also be affected.

SCHEDULE

  • Wednesday, November 11 - midnight (Eastern Time), through Monday, November 16 - 8 a.m. (Eastern Time)

AFFECTED SERVICES AND APPLICATIONS:

  • Pegasys - The application will not be available, and no tasks can be performed during this time.
  • Purchase Card Holders - The inability to create logs during this time frame will not be subject to the "Score Card."
  • CCRC - The application will not be available, and no tasks can be performed during this time.
  • VCSS - The application will not be available, and no tasks can be performed during this time.
  • CGE Travel - Authorizations and vouchers can be processed. No payments will be processed during this time. They will automatically be processed once Pegasys becomes available; therefore, no action will be required of travelers or approvers once the vouchers are submitted.
  • Business Objects - Since Pegasys will be unavailable from November 11-16, data in Business Objects universes that access Pegasys data will only contain data through November 10 until Pegasys operations resume. Universes that do not utilize Pegasys data will be unaffected.

QUESTIONS?

If problems persist beyond the scheduled dates and times, users are advised to contact the Financial Systems Service Desk at 1-866-450-6588 or via email at OCFOServiceDesk@gsa.gov

Pegasys News and Announcements
Attention PBS Pegasys Users:

The FY 2014 Appropriations Bill splits the PBS Building Operations budget account into two separate program codes (also known as budget activities). As a result of this change, a new program code (PG60) will be added to Pegasys effective Friday, August 8.

Pegasys transactions for PBS Building Operations will now be recorded with one of the following:

  • Program code (PG61) will continue to be used for Building Services transactions.
  • The new program code (PG60) will be used for Salaries and Expenses transactions.

Please review your financial transactions to ensure you comply with these changes beginning Friday, August 8.

Click to view a crosswalk, which identifies the transactions that should continue to use code PG61 and those that should use PG60.

If you have questions, please contact Dawn Stalter at dawn.stalter@gsa.gov or (202) 205-8921.

AGENCY CODE CHANGES IN PEGASYS DUE TO BAAR PHASE 2A IMPLEMENTATION

Pegasys now identifies an agency with a three-digit code. This transition to using a three-digit code to identify agencies is a part of the Government-wide Treasury Account Symbol Adjusted Balance System (GTAS) project and was part of the recent implementation of the Phase 2A Billings and Accounts Receivable (BAAR) module.

To find an agency’s new identification code, visit https://insite.gsa.gov/portal/content/513050, and review the document “2-Digit to 3-Digit Agency Code Crosswalk.”
