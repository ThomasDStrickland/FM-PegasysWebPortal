Financial Codes

Requests for adding or deactivating financial codes and the associated accounting and CGE/Concur templates, as well as changing code descriptions or titles should be submitted to SSOCode.request@gsa.gov using the Accounting Code Change Form.

Vendors

New vendor number requests, reactivation, changes for employees, FAS or Staff Offices should be submitted to the KC Finance Center.
New vendor requests, reactivation, changes for PBS vendors should be submitted to FW Finance Center.

Purchase Cards

New cardholder applications and changes to approving officials should be submitted to your Local Purchase Card Coordinator.

UPS/FedEx

UPS/FedEx accounts and changes to UPS/FedEx templates should be submitted to Maria Ellis.

Pegasys Access

For information about Pegasys access including user IDs, changes to roles, approvals, and security orgs, as well as changes to credit card reconcilers, please visit the Pegasys Access page.
